# GhostProver — Documentation Assets

[English](./README.md) | [简体中文](./README.zh-CN.md)

该目录包含 GhostProver 在文档、演示与展示材料中使用的静态媒体资源。

## 包含资源

| 文件 | 作用 |
|---|---|
| `ghostprover.jpeg` | GhostProver 主视觉素材 |
| `ghostprover_compliance.png` | 合规主题的辅助视觉素材 |
| `architecture.png` | 可用于展示的架构总览图片 |
| `prompt.png` | 单个 Prompt 决策流程图片 |
| `proof.png` | ZK 证明生命周期图片 |
| `0g-stack-flow.excalidraw` | 可编辑的 GhostProver + 0G 流程 Excalidraw 源文件 |

这些资源被专门放在 `docs/assets/` 下，以便让仓库根目录保持对源码与主文档的聚焦。
