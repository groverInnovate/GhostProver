# GhostProver — Examples

[English](./README.md) | [简体中文](./README.zh-CN.md)

该目录包含用于扩展 GhostProver 的示例配置文件，无需修改源码即可使用。

## 包含的文件

| 文件 | 作用 |
|---|---|
| `custom-registry.json` | 企业自定义模式与预设的示例注册表 |
| `.ghostprover.custom.example.json` | 演示如何加载自定义注册表的本地配置文件 |

## 适用场景

当你需要：

- 定义内部敏感数据模式
- 创建组织级合规预设
- 用非内置策略测试 GhostProver

就可以参考这些示例文件。

这些文件是参考资产，不是必须的运行时文件。
